# De Pátria para Pátria

## Descrição do Projeto

Este projeto é uma página web responsiva que apresenta histórias de pessoas de diferentes cidades ao redor do mundo. O site foi desenvolvido como parte do Sprint 5 do curso de Desenvolvimento Web da Tripleten, focando em design responsivo e boas práticas de CSS.

## Funcionalidades

- Design totalmente responsivo para desktop (1280px), tablet (768px) e mobile (320px)
- Seções organizadas com informações sobre diferentes cidades
- Implementação da metodologia BEM para organização do CSS
- Otimização de imagens para diferentes resoluções

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3
- Metodologia BEM para organização do CSS
- Flexbox
- CSS Grid
- Media Queries para responsividade

## Responsividade

O projeto utiliza dois breakpoints principais:

- **544px**: Transição entre mobile e tablet
- **1024px**: Transição entre tablet e desktop

O layout foi testado em resoluções intermediárias (470px, 665px, 999px, 1100px, 1500px) para garantir uma experiência suave em todos os dispositivos.
